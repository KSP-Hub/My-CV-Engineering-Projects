# Changelog

## [1.0.0] - 2025-12-25

### Добавлено
- Новый файл `CHANGELOG.md` для отслеживания изменений
- Полная документация в папке `docs/`:
  - `getting_started/getting_started.md` - инструкция по началу работы
  - `api_reference/api_reference.md` - документация по API
  - `troubleshooting/troubleshooting.md` - руководство по устранению неполадок
  - `contributing/contributing.md` - инструкции для разработчиков
- Единый `README.md` с объединенной документацией
- Ядро детекции лиц `face_detection_core.py` для устранения дублирования кода
- Конфигурационный класс `FaceDetectionConfig` для настройки параметров детекции

### Изменено
- Обновлен `README.md` с новой структурой документации
- Устранено дублирование кода между `face_detection.py` и `CV-010-app.py` путем вынесения общей логики в `face_detection_core.py`
- Переработаны пути к файлам для лучшей совместимости
- Обновлены инструкции по установке и использованию
- Улучшена структура проекта согласно рекомендациям

### Исправлено
- Исправлена проблема с импортом `base64` в `face_detection_core.py`
- Улучшена обработка путей к файлам каскада Хаара
- Улучшена обработка ошибок при загрузке изображений

### Удалено
- Устаревшие дублирующие файлы документации

## [0.1.0] - 2025-11-28

### Добавлено
- Инициализация проекта CV-010 Детекция лиц
- Реализация детектора лиц с использованием Haar cascades
- Консольное приложение `face_detection.py`
- Веб-приложение на Flask `CV-010-app.py`
- Базовые тесты
- Начальная документация в `README.md` и `README_COMBINED.md`
