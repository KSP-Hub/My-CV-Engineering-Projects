# Резюме проекта CV-011: Распознавание QR-кодов

## Общая информация
- **Название проекта**: CV-011: Распознавание QR-кодов
- **Дата завершения**: 29 декабря 2025
- **Статус**: Готов к релизу
- **Версия**: 1.0.0

## Описание проекта
Проект представляет собой веб-приложение для распознавания QR-кодов с использованием OpenCV и Flask. Пользователь может загружать изображения с QR-кодами, и приложение распознает их содержимое, отображает результат и позволяет скачать обработанное изображение с визуализацией распознанного кода.

## Реализованные функции
- Веб-интерфейс с возможностью загрузки изображений
- Распознавание QR-кодов с использованием OpenCV
- Визуализация результатов с отрисовкой рамок вокруг QR-кодов
- Скачивание обработанных изображений
- Прогресс-бар для визуальной индикации процесса обработки
- Обработка ошибок с подробными сообщениями
- Поддержка различных форматов изображений

## Технологический стек
- **Фронтенд**: HTML, CSS, JavaScript
- **Бэкенд**: Python, Flask
- **Обработка изображений**: OpenCV (cv2)
- **Дополнительно**: NumPy

## Структура тестирования
- **Unit-тесты**: Полное мокирование зависимостей (cv2.imread и QR-детектор)
- **Интеграционные тесты**: Работа с реальными изображениями
- **Разделение тестов**: 
  - tests/unit/ - модульные тесты
  - tests/integration/ - интеграционные тесты

## CI/CD Pipeline
- Настроен GitHub Actions для автоматической проверки кода
- Pipeline запускает unit-тесты и интеграционные тесты при push/pull request в ветки main и master
- Используется Python 3.8 для тестирования
- Процесс включает установку зависимостей и запуск pytest
