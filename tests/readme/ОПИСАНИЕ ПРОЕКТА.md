# My-CV-Engineering-Projects

## Краткое описание
Проект представляет собой портфолио для самопрезентации начинающего инженера по компьютерному зрению (CV engineer). Включает серию учебных проектов по обработке изображений, распознаванию объектов и компьютерному зрению на Python.

## Цели проекта
- Создать публичное портфолио для демонстрации навыков CV-инженера
- Реализовать серию проектов по компьютерному зрению
- Автоматизировать разработку с помощью ИИ-ассистента
- Настроить эффективный workflow разработки

## Технологии и инструменты
- Python
- OpenCV
- Flask
- Git / GitHub
- Git LFS
- Jupyter Notebook
- PyInstaller

## Требования к системе
- Python 3.7+
- Windows 10/11 или Linux
- 8+ GB RAM (рекомендуется 16+ GB)
- GPU с поддержкой CUDA (рекомендуется для нейросетевых проектов)

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/KSP-Hub/My-CV-Engineering-Projects.git
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Установите Git LFS:
   ```bash
   git lfs install
   ```

## Запуск
Запуск зависит от конкретного подпроекта. Для CV-010:
```bash
python CV-010_face_detection/CV-010-app.py
```

## Использование
Каждый подпроект представляет со~~~~бой законченное приложение или библиотеку по компьютерному зрению. Проекты можно использовать как учебные примеры или как основу для собственных разработок.

## Примеры
- CV-010: Веб-приложение для распознавания лиц
- CV-011: Распознавание QR-кодов
- CV-012: Фильтры изображений

## Тестирование
Для тестирования используйте встроенные тесты в папке `tests/`:
```bash
pytest tests/
```

## Проблемы и ограничения
- Ограничение GitHub на размер файлов (100 МБ)
- Требуется настройка Git LFS для больших бинарных файлов
- Некоторые проекты требуют значительных вычислительных ресурсов
