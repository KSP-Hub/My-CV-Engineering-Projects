## id: 10. `CV-010_face_detection/`

Проект по распознаванию лиц с использованием OpenCV и Haar cascades. Представляет собой веб-приложение на Flask, позволяющее загружать изображения, детектировать лица и визуализировать результат. Основные компоненты:

- `app.py` — веб-сервер Flask с REST API для обработки изображений
- `face_detection.py` — логика детекции лиц с использованием Haar cascades
- `real_time_detection.py` — распознавание лиц в реальном времени с веб-камеры
- `haarcascade_frontalface_default.xml` — предобученный каскад Хаара
- `templates/` — HTML шаблоны веб-интерфейса
- `static/` — папка для сохранения результатов обработки
