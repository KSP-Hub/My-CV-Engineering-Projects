## `CV-000-ai-assistant/`

Проект по настройке автономного AI-ассистента для помощи в написании кода на Python для задач компьютерного зрения. Реализован без зависимости от санкционных сервисов, с поддержкой оффлайн-работы и интеграцией с VS Code. Основные характеристики:

- Работает локально на Windows/Linux/macOS
- Оптимизирован под аппаратную конфигурацию Dell Precision 7920 Tower
- Специализация на CV-библиотеках (OpenCV, YOLO, PyTorch)
- Время отклика ≤ 10 секунд
- Поддержка русского языка интерфейса

## `CV-010_face_detection/`

Проект по распознаванию лиц с использованием OpenCV и Haar cascades. Представляет собой веб-приложение на Flask, позволяющее загружать изображения, детектировать лица и визуализировать результат. Основные компоненты:

- `app.py` — веб-сервер Flask с REST API для обработки изображений
- `face_detection.py` — логика детекции лиц с использованием Haar cascades
- `real_time_detection.py` — распознавание лиц в реальном времени с веб-камеры
- `haarcascade_frontalface_default.xml` — предобученный каскад Хаара
- `templates/` — HTML шаблоны веб-интерфейса
- `static/` — папка для сохранения результатов обработки

## `CV-011-QR-code-recognition/`

Проект по распознаванию QR-кодов с использованием OpenCV и веб-интерфейса на Flask. Будет включать:

- Возможность генерации QR-кодов
- Сохранение результатов обработки

## 4. `CV-012-image-filters/`

## 5. `CV-013-CNN-classification/`

Проект по классификации изображений с использованием сверточных нейронных сетей (CNN). Будет включать:

- Построение и обучение CNN модели на наборе данных CIFAR-10 или ImageNet
- Веб-интерфейс для загрузки изображений и классификации
- Визуализацию результатов с указанием вероятностей классов
- Возможность дообучения модели на пользовательских данных

## 6. `CV-014-transfer-learning/`

Проект по использованию transfer learning для классификации изображений. Будет включать:

- Использование предобученных моделей (ResNet, VGG, Inception)
- Адаптацию моделей под специфические задачи классификации
- Сравнение производительности различных архитектур

## 7. `CV-015-YOLO-detection/`

Проект по детекции объектов с использованием YOLO (You Only Look Once). Будет включать:

- Реализацию YOLO v5/v8 для детекции множества объектов
- Веб-интерфейс для обработки изображений и видеопотоков

## 8. `CV-016-U-Net-segmentation/`

Проект по сегментации изображений с использованием U-Net. Будет включать:

- Реализацию U-Net архитектуры для semantic segmentation
