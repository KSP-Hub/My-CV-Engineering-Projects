# Release Package for CV-000-ai-assistant

## Содержимое пакета релиза

Этот пакет содержит минимальный набор файлов для запуска и использования CV-000-ai-assistant:

- `scripts/` - все необходимые скрипты для управления и запуска
  - `start_cv_assistant.ps1` - запуск Ollama сервера с оптимизированной моделью
  - `run_switch.bat` - переключение между локальной и облачной моделью
  - `test_functionality.bat` - проверка работоспособности компонентов
- `docs/` - основная документация
  - `README.MD` - основное руководство по проекту
  - `requirements.txt` - зависимости

## Инструкция по использованию

### Требования
- PowerShell 5.1+
- Python 3.8+
- Ollama (установлен отдельно)

### Установка
1. Убедитесь, что Ollama установлен: https://ollama.com
2. Установите зависимости:
```bash
pip install -r docs/requirements.txt
```

### Запуск
1. Запустите Ollama сервер:
```powershell
scripts/start_cv_assistant.ps1
```
2. Переключитесь на нужную модель:
```cmd
scripts/run_switch.bat
```
3. Проверьте работоспособность:
```cmd
scripts/test_functionality.bat
```

## Ключевые функции

### 1. Автономная работа
- Полностью оффлайн-режим без зависимости от санкционных сервисов
- Оптимизация под локальное железо (Dell Precision 7920, Quadro RTX 4000)
- Поддержка русского языка

### 2. Управление моделями
- Автоматическое создание оптимизированной модели `cv-rtx4000` при первом запуске
- Переключение между локальной и облачной моделью через `run_switch.bat`
- Автоопределение доступности моделей

### 3. Интеграция с VS Code
- Полный доступ к файловой структуре проекта
- Контекстная память для долгосрочных целей
- Специализация на CV-библиотеках (OpenCV, PyTorch)

### 4. Надежность и проверка
- Скрипт `test_functionality.bat` для проверки всех компонентов
- Автоматическая проверка наличия необходимых файлов и путей
- Подробное логирование работы

### 5. Простота настройки
- Автоматизированная установка без "слётов"
- MVP-подход без излишеств
- Бесплатность и отсутствие платных API
